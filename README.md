Oferta Laboral 1 – Análisis
Portal: LinkedIn Jobs
Empresa: Konrad
Ubicación: San José, Costa Rica (Híbrido)
Puesto: Data Analyst

Nombre del puesto
Data Analyst
Analista encargado de analizar datos empresariales para generar información útil que apoye la toma de decisiones de negocio.

Tecnologías requeridas
Según la descripción del puesto, las principales tecnologías son:
•	Python
•	R
•	Bash scripting
•	SQL
•	Snowflake
•	Databricks
•	Amazon Redshift
•	Google BigQuery
•	Looker
•	Tableau
•	Power BI

Habilidades técnicas (Hard Skills)
Las habilidades técnicas identificadas en la oferta son:
•	Análisis de datos
•	Consulta y manejo de bases de datos con SQL
•	Programación en Python o R
•	Visualización de datos con herramientas de Business Intelligence
•	Creación de dashboards
•	Modelado de datos
•	Definición y seguimiento de KPIs
•	A/B Testing
•	Análisis causal
•	Validación de calidad de datos
•	Análisis exploratorio de datos

Habilidades blandas (Soft Skills)
Las habilidades blandas mencionadas en la oferta incluyen:
•	Pensamiento analítico
•	Atención al detalle
•	Comunicación clara (oral y escrita)
•	Trabajo en equipo
•	Colaboración con equipos multidisciplinarios
•	Capacidad de aprendizaje continuo
•	Curiosidad por el análisis de datos

Responsabilidades principales
Entre las responsabilidades del puesto se encuentran:
•	Analizar datos para identificar tendencias, patrones e insights
•	Construir y mantener dashboards y reportes
•	Traducir preguntas de negocio en requerimientos analíticos
•	Colaborar con data engineers, analistas y científicos de datos
•	Comunicar resultados mediante visualizaciones y presentaciones
•	Documentar métricas y procesos analíticos
•	Mejorar la calidad de los datos
•	Apoyar la creación de modelos de datos y reportes analíticos
•	Realizar seguimiento de KPIs

• Integración y consumo de APIs de datos
• Manejo de formatos de datos como JSON
• Desarrollo y monitoreo básico de modelos de machine learning

• Pensamiento analítico

• Analizar conjuntos de datos para identificar tendencias y patrones
• Mantener bases de datos, dashboards y reportes actualizados
• Crear visualizaciones de datos para apoyar la toma de decisiones
• Colaborar con diferentes equipos para apoyar iniciativas estratégicas
